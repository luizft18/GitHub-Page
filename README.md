# GitHub-Page
Criando o site do portfólio.

Este repositório foi criado como parte da Entrega Intermediária da disciplina de Bootcamp I, com o objetivo de desenvolver habilidades de manuseamento da plataforma GitHub e agregar informações à um portfólio profissional.

Incluidos neste repositório estão: 
• O arquivo index.html contendo o código da página;
• As pastas contendo os arquivos em JavaScript e CSS do template utilizado para a GitHub Page nele incluída;
• Os aqruivos utilizados para o preenchimento e composição da página.

Programas utilizados para o desenvolvimento dos projetos:
• Google Docs;
• Google Slides;
• Google Sites;
• HTML;
• JavaScript;
• CSS.
